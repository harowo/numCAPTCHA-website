Introduction

What is numCAPTCHA?

numCAPTCHA is an open-source CAPTCHA service that protects your website from spam bots.
numCAPTCHA takes user-submitted equations and stretch or manipulate the numbers and operators, making them simple to solve for human, but difficult to recognise for bots.
By adding numCAPTCHA to your website, you can add another layer of security to block unwanted traffic.

Usage

How to use numCAPTCHA?

numCAPTCHA is easy to set up and use.
We have two libraries that can be use to implement numCAPTCHA into your website.

JavaScript

In your index.html, numCaptcha dynamically renders in a <div> tag with the id of captcha-wrapper
<body>
    <div id="captcha-wrapper"></div>
</body>
You also need to import the minified script using a script tag
<script src="https://numcaptcha.s3.amazonaws.com/lib/numCaptchaJS/numcaptcha-js.min.js"></script>
After importing the script, you would have access to the useNumCaptcha() function in the global windows object. The function returns a promise which resolves with the value of true when user successfully completes numCAPTCHA, else it would reject with the value of false when user is identified as a bot.
window
        .useNumCaptcha()
        .then((res) => {
                // do something onSuccess
        })
        .catch((err) => {
                // do something onFailed
        });

Vue.js

In your vue app, install the plugin from npm as a dependency.
npm i --save numcaptcha-vue
In your main.js, call the Vue.use() global method to use the plugin.
// firstly, import and use Font Awesome
import { library } from '@fortawesome/fontawesome-svg-core';
import { fas } from '@fortawesome/free-solid-svg-icons';
import { FontAwesomeIcon } from '@fortawesome/vue-fontawesome';

library.add(fas);
Vue.component('font-awesome-icon', FontAwesomeIcon);

// then, import and use numCaptcha
import NumCaptcha from 'numcaptcha-vue';

Vue.use(NumCaptcha, { FontAwesomeIcon, fas });

/* ... rest of main.js ... */
After importing the plugin, you can use it anywhere in your vue application using the <num-captcha /> component.
<num-captcha @status="success | failed" />

Submission Rules

Since numCAPTCHA is open-sourced, we accept user submission of their own equations to expand our library.
To protect the integrity of numCAPTCHA, submissions must meet the requirements below.
  1. Image must be in .jpg format
  2. Image dimensions must be within a height of 55 pixels and length of 135 pixels
  3. Stroke in image must be one pixel thick only
  4. The equation must be simple enough to solve. If the difficulty is too high, the submission will be rejected
After meeting the above requirements, feel free to submit here.
Note that submissions go through a reviewing process.
